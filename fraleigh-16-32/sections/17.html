Definitions 4
  • \(\operatorname{Fix}(g)=\{x\in X\mid gx=x\}\). 한 group element가 고정하는 객체들의 집합.
  • The number of essentially different objects under a group action is the number of orbits.
  • Conjugation action is \(g*x=gxg^{-1}\). Its orbits are conjugacy classes.
  • The centralizer \(C_G(x)\) is the stabilizer of x under conjugation.
Theorems 3

Burnside lemma

If a finite group G acts on a finite set X, then the number of orbits is \(\frac{1}{|G|}\sum_{g\in G}|\operatorname{Fix}(g)|\).

Proof spine. Count \(S=\{(g,x)\mid gx=x\}\). By g first, |S|=sum |Fix(g)|. By orbit first, each orbit contributes |G| using orbit-stabilizer.

Use. up to symmetry 문제의 기본 공식.

Class equation

For a finite group G under conjugation, \(|G|=|Z(G)|+\sum_i [G:C_G(a_i)]\), where a_i run over representatives of noncentral conjugacy classes.

Proof spine. Conjugation orbits of central elements have size 1. Other orbit sizes are centralizer indices.

Use. p-group의 center가 nontrivial임을 보이는 표준 입구다.

p-group center theorem

If |G|=p^n with p prime and n>0, then \(Z(G)\) is nontrivial.

Proof spine. In the class equation, every noncentral conjugacy class has size divisible by p. Since |G| is divisible by p, |\(Z(G)\)| is divisible by p.

Use. Sylow 전 단계에서 자주 쓰는 group action 계산.

Propositions 2

Fixed colors under a rotation

For necklace colorings, a rotation by k positions fixes exactly \(c^{\gcd(n,k)}\) colorings when c colors and n beads are used.

Proof spine. The rotation splits positions into gcd(n,k) cycles, and each cycle must be monochromatic.

Conjugacy class size

The size of the conjugacy class of a is [G:C_G(a)].

Proof spine. This is orbit-stabilizer for the conjugation action.

Examples 5

Three-bead necklaces with two colors

C3 acts on 2^3 colorings. The identity fixes 8, each nontrivial rotation fixes 2. Burnside gives (8+2+2)/3=4 necklaces.

Use. fixed set을 직접 세는 연습.

D4 colorings of a square

D4 has rotations and reflections. Count colorings fixed by each symmetry type, then average over eight symmetries.

Use. 대칭을 conjugacy type별로 묶으면 계산이 짧아진다.

Conjugation in S3

S3 has conjugacy classes: identity, three transpositions, two 3-cycles. This is exactly the orbit decomposition under conjugation.

Use. conjugacy class는 추상 정의가 아니라 group action의 orbit이다.

Round-table seating

Circular seating is counted by quotienting ordinary arrangements by the rotation action.

Use. If all n people are distinct, rotations identify n linear arrangements, giving (n-1)! circular arrangements.

Cube face markings

A cube-marking problem is a Burnside problem for the 24-element rotation group of the cube.

Use. It is the canonical example where fixed counts are grouped by rotation type.

Exercises 6

Burnside warm-up

Count binary bracelets of length 4 under rotations only.

Proof spine. Use C4. Fixed counts: identity 16, 90-degree rotations 2 each, 180-degree rotation 4.

Class equation

Write the class equation of S3.

Proof spine. Use class sizes 1, 3, 2.

Center of order p^2 group

Use the p-group center theorem to show every group of order p^2 is abelian.

Proof spine. After \(Z(G)\) is nontrivial, consider whether its order is p or p^2 and use G/\(Z(G)\) cyclic.

Conjugacy stabilizer

For a transposition in S4, compute the size of its conjugacy class via the centralizer.

Proof spine. There are six transpositions; centralizer size must be 24/6=4.

Table seating

How many ways can seven distinct people sit around a round table up to rotation?

Proof spine. 6!.

Fixed-point table

For a square with two colors on vertices, make the full Burnside fixed-point table for D4.

Proof spine. Group the eight symmetries by type: identity, rotations, and reflections.

Proof Patterns 2
  • Counting orbits usually means building the set of pairs (g,x) with gx=x.
  • Conjugacy problems become stabilizer problems once you write the action explicitly.
