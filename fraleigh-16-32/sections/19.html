Definitions 3
  • A zero divisor is a nonzero a for which some nonzero b satisfies ab=0.
  • An integral domain is a commutative ring with unity 1 != 0 and no zero divisors.
  • The characteristic of a ring with unity is the least positive n with n*1=0, if it exists; otherwise it is 0.
Theorems 5

Cancellation criterion

In a ring, cancellation by every nonzero element is valid exactly when the ring has no zero divisors.

Proof spine. If ac=bc, then a(c-b)=0. Without zero divisors and a != 0, c=b. Conversely, a zero divisor kills cancellation.

Finite integral domain is field

Every finite integral domain is a field.

Proof spine. For a nonzero a, the map x -> ax is injective by cancellation. A finite injective map is surjective, so ax=1 has a solution.

Characteristic of a domain

The characteristic of an integral domain is either 0 or a prime number.

Proof spine. If char is mn with 1<m,n<char, then (m*1)(n*1)=0 gives zero divisors.

Every field is a domain

Every field is an integral domain.

Proof spine. If ab=0 and a != 0, multiply by a^{-1} to get b=0.

Z_p field criterion

\(\mathbb Z_p\) is a field exactly when p is prime.

Proof spine. Prime p gives no zero divisors; if n is composite, \(\mathbb Z_n\) has zero divisors.

Propositions 2

Z_n zero divisors

In \(\mathbb Z_n\), a nonzero class is a zero divisor iff gcd(a,n) != 1.

Proof spine. If d=gcd(a,n)>1 then a*(n/d)=0 mod n. If \(\gcd(a,n)=1\), multiplication by a is invertible.

Subrings of domains

A subring with the same nonzero unity inside an integral domain has no zero divisors.

Proof spine. A zero divisor relation in the subring would be one in the domain.

Examples 4

Z and Z_p

Z is a domain but not a field. \(\mathbb Z_p\) is a finite domain and therefore a field.

Use. infinite domain과 finite domain의 차이를 고정한다.

Z_6

In \(\mathbb Z_6\), 2*3=0 while 2 and 3 are nonzero. Hence \(\mathbb Z_6\) is not a domain.

Use. composite modulus gives zero divisors.

Polynomial domain

If D is a domain, D[x] is again a domain.

Use. leading terms multiply without disappearing.

Matrix zero divisors over a field

Even if F is a field, \(M_2(F)\) has zero divisors such as nonzero singular matrices.

Use. domain requires commutativity and no zero divisors; matrix rings fail both patterns.

Exercises 5

Cancellation

In an integral domain, prove that ax=ay and a != 0 imply x=y.

Proof spine. Move to a(x-y)=0.

Characteristic

Show a finite field has prime characteristic.

Proof spine. A field is a domain.

Z_15

List the zero divisors of \(\mathbb Z_{15}\).

Proof spine. Nonzero nonunits: 3,5,6,9,10,12.

Finite domain

Let D have 10 elements and no zero divisors. Explain why every nonzero a has inverse.

Proof spine. Use injective x->ax.

Field to domain

Write the one-line proof that a field has no zero divisors.

Proof spine. Multiply ab=0 by a^{-1} when a is nonzero.

Proof Patterns 2
  • Finite + injective = surjective is the standard engine.
  • Characteristic arguments turn composite n into (m*1)(k*1)=0.
