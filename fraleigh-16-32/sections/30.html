Definitions 5
  • A vector space over F is an abelian group V with scalar multiplication by F satisfying the usual distributive and identity laws.
  • A set spans V if every vector is a finite linear combination of its elements.
  • A set is linearly independent if only the trivial finite linear combination gives 0.
  • A basis is a spanning linearly independent set.
  • The dimension of a finite-dimensional vector space is the size of a basis.
Theorems 5

Spanning set contains a basis

Every finite spanning set of a vector space contains a basis.

Proof spine. Remove dependent vectors one at a time without changing the span until independence remains.

Independent set bound

In a finite-dimensional vector space, any linearly independent set has at most as many elements as any spanning set.

Proof spine. Exchange one independent vector into a spanning set at a time.

Degree of simple extension

If alpha is algebraic over F with minimal polynomial degree n, then [\(F(\alpha)\):F]=n and {\(1,\alpha,\ldots,\alpha^{n-1}\)} is a basis.

Proof spine. Section 29 gives unique expression; that is exactly basis existence and uniqueness.

Vector space zero laws

In a vector space, 0v=0, a0=0, and (-a)v=-(av).

Proof spine. The proof mirrors ring zero laws using additive cancellation.

Finite-dimensional space has finite basis

Every finite-dimensional vector space has a finite basis.

Proof spine. Start from a finite spanning set and remove dependent vectors.

Propositions 2

All finite bases have same size

Any two finite bases of V have the same number of elements.

Proof spine. Apply the independent set bound in both directions.

Finite-dimensional subspace bound

A linearly independent set in an n-dimensional space has at most n vectors.

Proof spine. Use any basis as a spanning set.

Examples 4

R^n over R

The standard unit vectors form a basis, so dim_R R^n=n.

Use. baseline vector-space model.

C over R

C has basis {1,i} over R, so \([\mathbb C:\mathbb R]=2\).

Use. extension degree as dimension.

Q(sqrt2) over Q

Every element has form a+b \(\sqrt2\) with a,b in Q; basis {1,\(\sqrt2\)}.

Use. minimal polynomial degree 2 becomes vector-space degree 2.

Extension as vector space

Every extension field E of F is a vector space over F by using field multiplication as scalar multiplication.

Use. This is why field degree is a dimension.

Exercises 5

Basis check

Show {1,\(\sqrt2\)} is linearly independent \(\text{over }\mathbb Q\).

Proof spine. a+b \(\sqrt2\)=0 with rational a,b implies \(\sqrt2\) rational unless b=0.

Dimension

Find [\(\mathbb Q(\sqrt[3]{2})\):Q].

Proof spine. Minimal polynomial \(x^3-2\) is irreducible \(\text{over }\mathbb Q\).

Span

Explain why \(F[x]\) is infinite-dimensional over F.

Proof spine. The monomials 1,x,x^2,... cannot be spanned by finitely many of them.

Exchange

If V is spanned by 3 vectors, can it contain 4 linearly independent vectors?

Proof spine. No.

Zero scalar

Prove 0v=0 in a vector space.

Proof spine. Use (0+0)v=0v+0v and cancel.

Proof Patterns 2
  • Basis proofs split into spanning and independence.
  • Extension degree questions become minimal polynomial degree questions for simple algebraic extensions.
