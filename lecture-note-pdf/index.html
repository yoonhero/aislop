Lecture Note PDF

추상선형대수학 1.1-2.3 원문을 한 권의 PDF로 재구성

원문 표현은 가능한 한 유지하고, 읽기 편한 레이아웃과 절별 연습문제, `★ 중요` 표시, `추가 문제`를 함께 제공한다. HTML 미리보기와 PDF를 함께 제공한다.

PDF 열기 HTML 미리보기 원고 보기

포함 범위

  • 1.1 벡터공간, 부분공간, 선형결합
  • 1.2 기저의 존재성과 차원
  • 1.3 선형변환, 커널, 이미지
  • 1.4 몫과 코커널
  • 2.1 동형사상
  • 2.2 동형정리
  • 2.3 직곱, 사영, 쌍대공간

구성 원칙

  • 원문 내용을 가능한 한 유지하고 서식만 정리
  • 원문 연습문제를 절별로 모두 포함
  • 중요 문제는 `★ 중요`, 보강 문제는 `추가 문제`로 분리

빌드

python3 scripts/build.py

원문 사이트를 기반으로 작성했고, 산출물은 dist/에 생성된다.
