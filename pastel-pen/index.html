Pastel
Pen

STAEDTLER-ish soft highlighting for Firefox: local-only, no account, no sync, no publishing pipeline. Select text, save it, review it later.

Load It

  1. Open Firefox about:debugging#/runtime/this-firefox.
  2. Click Load Temporary Add-on.
  3. Select pastel-pen/extension/manifest.json.

The extension stores highlights in Firefox storage.local. Open the toolbar popup or extension options to review, search, export, import, and delete saved marks.

Edge Cases

Dynamic pages

A debounced observer replays restoration when ChatGPT, Naver Blog, Tistory, and infinite articles mutate after load.

Rich selections

The marker wraps text nodes instead of surrounding whole DOM ranges, avoiding broken links, code spans, tables, and inline math.

Duplicate quotes

Context and distance from the original text position rank candidates when the same sentence appears repeatedly.

Local review

The review page is an extension page backed by local Firefox storage, with export/import as the no-cloud escape hatch.
