## Design Summary

The final direction intentionally avoids a dense poster collage. It follows a clean shirt/hoodie graphic structure:

- image-rendered `Goodbye, BOJ` title
- pale mint rectangular block
- centered Jank bust
- bold simplified segment-tree icon
- bold simplified DP-table icon
- large negative space for hoodie-back readability

See `character-development/poster-process.md` for the full production process.
