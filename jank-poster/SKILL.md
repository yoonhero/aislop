# 포스터 비주얼 구성 핵심 정리

## 1. 포스터는 “정보 목록”이 아니라 “한 장면”이다

포스터의 중심은 설명이 아니라 **강한 시각적 장면**이어야 한다.

나쁜 흐름:

```text
넣어야 할 정보 정리 → 박스/배지/문구로 배치 → 전단지처럼 보임
```

좋은 흐름:

```text
강한 장면 구성 → 분위기 형성 → 필요한 정보만 최소한으로 배치
```

---

## 2. 시각적 위계는 7 : 2 : 1

포스터 안의 요소들은 비슷한 중요도로 놓이면 안 된다.

```text
7 = 메인 비주얼
2 = 제목 / 핵심 문구
1 = 세부 정보
```

### 7 — 메인 비주얼

가장 먼저 보여야 하는 것.

예시:

- 인물
- 캐릭터
- 사건의 순간
- 상징적 오브젝트
- 배경 세계관

### 2 — 제목 / 핵심 메시지

두 번째로 보여야 하는 것.

예시:

- 행사명
- 작품명
- 캠페인명
- 메인 슬로건

제목은 설명문이 아니라 **로고처럼** 보여야 한다.

### 1 — 세부 정보

마지막에 읽혀도 되는 것.

예시:

- 날짜
- 장소
- 신청 방법
- 주최
- 추가 공지

세부 정보가 포스터의 주인공이 되면 안 된다.

---

## 3. 텍스트는 줄일수록 강하다

포스터에서 텍스트는 많아질수록 힘이 약해진다.

좋은 텍스트:

```text
짧다
굵다
읽는 순서가 분명하다
역할이 명확하다
```

나쁜 텍스트:

```text
모든 것을 설명한다
비슷한 크기의 문구가 많다
말풍선, 배지, 박스가 많다
각 문구가 서로 경쟁한다
```

특히 상품, 혜택, 이벤트성 요소는 직접 문구로 쓰면 광고 배너처럼 보이기 쉽다.

---

## 4. 직접 말할 것과 암시할 것을 구분한다

### 직접 말해도 되는 것

- 행사명
- 날짜
- 장소
- 참가 대상
- 모집 여부
- 기본 공지

### 이미지로 암시하는 것이 좋은 것

- 상품이 있음
- 특별한 보상이 있음
- 규모감이 있음
- 경쟁 분위기
- 축제감
- 유머
- 긴장감

예를 들어 “상품 있음”을 말하고 싶다면,

```text
상품 준비 완료!
참가자 전원 선물 증정!
1등 상품 공개!
```

보다,

```text
빛나는 상자
트로피 실루엣
하늘에서 떨어지는 오브젝트
무대 가장자리의 선물상자
```

처럼 보여주는 편이 자연스럽다.

---

## 5. 오브젝트는 장면 안에 녹아들어야 한다

새 오브젝트를 추가할 때 핵심 질문은 이것이다.

```text
이 물체가 원래 이 장면 안에 있었던 것처럼 보이는가?
```

자연스러운 오브젝트의 조건:

- 원근이 맞음
- 빛 방향이 맞음
- 그림자와 하이라이트가 맞음
- 크기가 적절함
- 메인 피사체를 방해하지 않음
- 설명 없이도 의미가 전달됨

부자연스러운 오브젝트의 특징:

- 스티커처럼 붙어 있음
- 텍스트 배지로 설명됨
- 조명과 질감이 다름
- 크기가 과함
- 화면 중앙을 불필요하게 차지함

---

## 6. 시선 흐름을 설계한다

포스터는 보는 순서가 있어야 한다.

안정적인 흐름:

```text
상단 분위기 → 중앙 메인 피사체 → 제목 → 하단 정보
```

또는:

```text
중앙 인물 → 주변 사건 → 제목 → 날짜/신청 정보
```

시선을 유도하는 장치:

- 빛줄기
- 대각선 구도
- 원근선
- 인물의 시선
- 큰 오브젝트에서 작은 오브젝트로 이어지는 리듬
- 제목의 방향성

눈이 여기저기 튀면 포스터가 난잡해진다.

---

## 7. 여백은 비워진 공간이 아니라 구조다

여백은 낭비가 아니다.

여백의 역할:

- 메인 피사체를 돋보이게 함
- 제목을 읽히게 함
- 장면에 스케일을 줌
- 복잡한 요소 사이에 숨 쉴 공간을 만듦

비어 보인다고 계속 채우면 포스터가 무너진다.

---

## 8. 정보 박스는 최소화한다

박스는 정보를 정리해주지만, 많이 쓰면 전단지처럼 보인다.

박스를 써도 되는 곳:

- 날짜/장소
- 신청 정보
- 하단 정보 바
- 주최 정보

박스를 피하는 것이 좋은 곳:

- 상품 설명
- 특별상 설명
- 분위기 설명
- 홍보성 강조 문구
- 감탄형 문구

분위기는 박스가 아니라 이미지로 만든다.

---

## 9. 색은 역할을 나눠야 한다

색은 많다고 좋은 것이 아니다.  
각 색에 역할이 있어야 한다.

```text
배경색 = 전체 분위기
주요색 = 제목 / 브랜드 / 행사 정체성
강조색 = 시선 유도
중립색 = 정보 정리
```

모든 색이 동시에 강하면 아무것도 강조되지 않는다.

---

## 10. 강조는 하나만 한다

강조 요소가 많으면 포스터가 산만해진다.

동시에 강조하면 안 되는 것:

- 제목
- 캐릭터
- 상품
- 날짜
- 참가 문구
- 특별상
- 배경 오브젝트

보통 최우선 강조는 다음 둘이다.

```text
1. 메인 비주얼
2. 제목
```

나머지는 보조 역할로 내려야 한다.

---

## 11. 배경 오브젝트는 리듬 있게 배치한다

오브젝트를 여러 개 넣을 때는 균등하게 뿌리면 안 된다.

좋은 배치:

```text
큰 것 1개
중간 것 2~3개
작은 것 여러 개
멀수록 작고 흐림
방향성이 통일됨
```

나쁜 배치:

```text
비슷한 크기가 반복됨
화면 곳곳에 균등하게 흩어짐
원근이 없음
메인 피사체 주변을 침범함
```

---

## 12. 문구를 이미지로 번역한다

포스터가 자연스러워지려면 문구를 시각 요소로 바꾸는 습관이 필요하다.

```text
상품 있음 → 빛나는 상자, 리본, 트로피 실루엣
특별상 있음 → 독특한 오브젝트 하나
대회 느낌 → 무대, 조명, 관중석, 대진표, 전장
긴장감 → 역광, 낮은 카메라, 대각선 구도
축제감 → 색종이, 배너, 밝은 조명
모집 중 → 하단의 짧은 CTA 문구
```

말로 설명하지 말고 장면 속에 넣는다.

---

## 13. 하단은 정리만 담당한다

하단은 새로운 사건을 만드는 곳이 아니라 정보를 정리하는 곳이다.

좋은 하단:

```text
날짜
장소
신청/공지 문구
주최
```

나쁜 하단:

```text
상품 카드
혜택 목록
긴 설명
추가 슬로건
복잡한 아이콘 여러 개
```

하단이 복잡해지면 전체가 광고 배너처럼 변한다.

---

## 14. 포스터가 난잡해지는 원인

대부분의 실패 원인은 다음 중 하나다.

```text
1. 텍스트가 너무 많다
2. 강조 요소가 너무 많다
3. 오브젝트가 스티커처럼 붙어 있다
4. 혜택/상품을 너무 직접적으로 말한다
5. 하단 정보 영역이 과하다
6. 여백을 못 참고 채운다
7. 색과 효과가 전부 강하다
8. 제목보다 보조 문구가 더 튄다
```

---

## 15. 최종 점검 질문

포스터를 만들고 나서 다음을 확인한다.

```text
멀리서 봤을 때 가장 먼저 보이는 것은 무엇인가?
제목은 2초 안에 읽히는가?
세부 정보가 메인 비주얼을 방해하지 않는가?
텍스트 없이도 분위기가 전달되는가?
상품/혜택이 너무 노골적으로 보이지 않는가?
오브젝트들이 같은 장면 안에 있는 것처럼 보이는가?
하단이 전단지처럼 복잡하지 않은가?
```

---

## 핵심 요약

좋은 포스터는 정보를 많이 넣는 것이 아니라,  
**강한 장면 하나를 만들고 필요한 정보만 남기는 것**이다.

```text
메인 비주얼 7
제목 2
정보 1
```

가장 중요한 원칙:

> 말할 수 있는 것도 가능하면 보여준다.  
> 보여줄 수 없는 것만 짧게 말한다.
