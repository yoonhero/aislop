macOS experiment

Desktop Kanban

Finder Desktop icon positions를 kanban 상태로 해석하고, 반대로 카드 이동이 실제 Desktop icon 위치를 바꾸는 로컬 macOS 오버레이 앱입니다.

Core Behavior

기본은 click-through overlay입니다. 편집 모드에서만 drag and drop을 받고, Finder에서 직접 옮긴 아이콘도 다시 읽어서 상태를 반영합니다.

State

선택한 Desktop 항목만 bookmark로 추적하고, `Done` 진입 시 기록을 남깁니다. 기록과 보드 상태는 Application Support JSON으로 저장됩니다.

Run

cd /Users/ysh/Desktop/fun/aislop/desktop-kanban
swift build
swift run DesktopKanban

Notes

  • Finder Automation 권한이 필요합니다.
  • Desktop 정렬이 `Sort By: None`이 아니면 아이콘 이동은 차단됩니다.
  • 현재 버전은 주 모니터, 현재 Desktop space만 대상으로 합니다.
